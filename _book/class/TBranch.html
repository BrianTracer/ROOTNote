TBranch+

TBranch

继承 TNamed , TAttFill

A TTree object is a list of TBranchs.

A TBranch describes the branch data structure and supports :

  • the list of TBaskets (branch buffers) associated to this branch.
  • the list of TLeaves (branch description)
